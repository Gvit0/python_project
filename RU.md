## recursive_function_x_y_sum_search.py
Этот код определяет рекурсивную функцию F(x, y, n), которая принимает три аргумента: x, y, n.  Функция проверяет, больше ли сумма x и y 39, когда n равно 1. Если n не равно 1, функция рекурсивно вызывает себя, при этом x или y обновляются как сумма текущего значения и самого себя, в зависимости от  какое значение больше.

Код также инициализирует пустой список m для хранения сумм x и y, для которых функция F возвращает True.  Список m заполняется путем перебора всех возможных значений x и y от 0 до 19 и проверки, возвращает ли F(x, y, 0) значение True.

 Наконец, код печатает минимальное значение из списка m.  Этот код можно использовать для нахождения минимальной суммы x и y, для которой функция F возвращает значение True, когда n равно 0.

 Таким образом, этот код определяет рекурсивную функцию F, которая проверяет, превышает ли сумма двух чисел 39 при выполнении определенного условия, а затем находит минимальную сумму этих двух чисел, для которых функция возвращает True, когда n равно 0.

## petya_strategy.py 
Код Python, решающий проблему игры с камнями.

 описание проблемы
 Два игрока, Петя и Вася, играют в игру с двумя кучками камней.  В первой куче 12 камней, во второй — S камней.  Игроки по очереди убирают либо все камни из одной кучки, либо равное количество камней из обеих кучек.  Выигрывает тот игрок, который уберет последний камень.

 Цель
 Цель — найти такое минимальное значение S, при котором Петя не сможет выиграть партию за один ход, а Вася сможет выиграть партию с первого хода, независимо от хода Пети.
